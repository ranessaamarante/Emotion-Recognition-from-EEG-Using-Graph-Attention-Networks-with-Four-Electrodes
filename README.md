# Emotion-Recognition-from-EEG-Using-Graph-Attention-Networks-with-Four-Electrodes
EEG emotion recognition using Graph Attention Networks with 4 electrodes (FT7, FT8, P7, P8) on the SEED dataset. Features: differential entropy over delta and theta bands. Evaluation: Leave-One-Subject-Out (LOSO), 94.85% mean accuracy.
